# 📈 Stock Portfolio Tracker

## Description
A Python-based Stock Portfolio Tracker that allows users to manage stock investments using live stock prices from Yahoo Finance.

## Features

- Add Stocks
- View Portfolio
- Sell Stocks
- Portfolio Summary
- Live Stock Prices
- Profit/Loss Calculation
- Transaction History
- Export Portfolio to CSV

## Technologies Used

- Python
- JSON
- yfinance
- pandas

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
